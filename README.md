# 탄소배출 감소를 위한 컴플라이언스 가이드 제공
## 팀명 : Guardians
### ◾ 프로젝트 주제 및 개요
  - **주제**  
    국가 온실가스 종합 정보 데이터를 활용한 기업별 탄소 배출 현황을 제시하여 기업과 국민의 탄소중립 실현 및 탄소 배출 감소 방향성을 제시한다.
  - **목적**  
    국가 온실가스 배출량에 대한 정보를 제공함으로써 기업과 국민의 실질적 탄소중립 달성 및 탄소 배출 감소 기여에 목적으로 한다.
